# Paddleflow用户使用手册
Please select the user documentation in your preferred language:
- [简体中文](zh_cn)
- [English](en)

```mermaid
flowchat
st=>start: 开始框
op=>operation: 处理框
cond=>condition: 判断框(Yes/No)
sub1=>subroutine: 子流程
io=>inputoutput: 输入输出
e=>end: 结果
st->op(right)->cond
cond(yes)->io(right)->e
cond(no)->sub1(right)->op
```
